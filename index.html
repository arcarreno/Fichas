Bienvenido Administrador

Te guiaremos por las principales funciones del sistema.

Paso 1: Panel de Administración

Aquí podrás gestionar usuarios, fichas y reportes.

Crear Nuevo Usuario

Para agregar un nuevo usuario al sistema:

  • Haz clic en el botón "Usuarios"
  • Selecciona "Crear Usuario"
  • Completa el formulario y guarda

Gestionar Usuarios

Administra los usuarios existentes:

  • Visualiza la lista de usuarios
  • Elimina usuarios si es necesario

Creación de Fichas

Las fichas se crean en 4 pasos sencillos:

  • Info Básica
  • Ubicación en Mapa
  • Infraestructura
  • Subir Archivos PDF

Eliminar Fichas

Para eliminar una ficha:

  • Busca la ficha en la lista
  • Haz clic en el icono de eliminar
  • Confirma la acción

Editar Fichas

Modifica la información de las fichas:

  • Selecciona la ficha a editar
  • Actualiza los campos necesarios
  • Guarda los cambios

Exportar Datos

Exporta tus fichas a diferentes formatos:

  • PDF para impresión
  • Excel para análisis
  • Otros formatos disponibles

Reportes Generales

Genera reportes detallados del sistema:

  • Reportes de actividad
  • Estadísticas de fichas
  • Historial de cambios

Bienvenido Revisor

Te explicamos cómo usar el sistema como revisor.

Tu rol: Revisar y validar las fichas técnicas.

  • Visualiza todas las fichas asignadas
  • Añade comentarios y observaciones
  • Valida la información de las fichas

Comentarios y Observaciones

Añade comentarios a las fichas:

  • Selecciona una ficha
  • Escribe tu observación
  • Guarda el comentario

Modificar Fichas

Como revisor puedes modificar fichas:

  • Edita la información válida
  • Actualiza el estado de la ficha
  • Guarda los cambios realizados

Exportar Datos

Exporta las fichas revisadas:

  • PDF para revisión
  • Excel para informes

Reportes Generales

Genera reportes de tu actividad:

  • Historial de revisiones
  • Estadísticas de aprobación

Bienvenido Visor

Te explicamos cómo usar el sistema como visor.

Tu rol: Consultar y visualizar fichas técnicas.

  • Visualiza todas las fichas técnicas
  • Consulta la información de las fichas
  • Genera reportes de consulta

Comentarios

Consulta los comentarios de las fichas:

  • Selecciona una ficha
  • Revisa las observaciones

Exportar Datos

Exporta las fichas consultadas:

  • PDF para impresión
  • Excel para análisis

Reportes Generales

Genera reportes de consulta:

  • Historial de visualizaciones
  • Estadísticas de acceso
